# Porchlight Pizza

## What this is
Portfolio / reference Next.js site for Porchlight Pizza — a local pizza restaurant. Shipped client site kept as portfolio proof.

## Brand notes
- Porchlight theme — warm, neighborhood, family-friendly.
- Copy leans into community, hand-tossed, fresh ingredients.

## Gotchas
- Menu and hours are the most frequently-edited content — check they match the real restaurant before pushing.
- Online ordering (if present) is a third-party integration — don't mock it.
